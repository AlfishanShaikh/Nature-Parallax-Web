## 🚀 How to Use
1. Clone the repository:
   ```sh
   git clone https://github.com/AlfishanShaikh/Nature-Parallax-Web.git
   ```
2. Open the project folder:
   ```sh
   cd Nature-Parallax-Web
   ```
3. Open `index.html` in your browser.
